# The Enchanted Oven

The Enchanted Oven is a modern web application designed to bring magic to your baking sessions. Harnessing the power of cutting-edge technologies like **React**, **Vite**, and **Tailwind CSS**, this project promises a fast, flexible, and responsive user experience.

## Features
- Fast development with Vite.
- Elegant and customizable design using Tailwind CSS.
- Modular and scalable codebase with React.

---

## Installation

Follow these steps to quickly set up the project locally:

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Micjeal/the-enchanted-oven.git
   cd the-enchanted-oven
   ```

2. **Install Dependencies**
   Make sure you have Node.js and npm (or yarn) installed. Then, run the following command:
   ```bash
   npm install
   ```
   or, if you prefer Yarn:
   ```bash
   yarn install
   ```

3. **Start the Development Server**
   To launch the application in development mode, use the following command:
   ```bash
   npm run dev
   ```
   or with Yarn:
   ```bash
   yarn dev
   ```

4. **Open in Your Browser**
   Once the server is running, open your browser and navigate to:
   ```
   http://localhost:5173
   ```

## Usage

1. **Navigating the Application**
   - Explore the `Recipes` section for a wide variety of magical recipes.
   - Use the `Ingredients` manager to customize your baking needs.

2. **Customization**
   - The `Tailwind CSS` config provides options to easily tweak the design according to your preferences. Edit the `tailwind.config.js` file to get started.

3. **Building the Project**
   To create a production build, run:
   ```bash
   npm run build
   ```
   or with Yarn:
   ```bash
   yarn build
   ```
